# FRAS - Face Recognition Attendance System  
*Final Year Project | Python + OpenCV + Flask*  

---

## 📌 **Project Overview**  
FRAS is a **web-based attendance system** that uses facial recognition to mark attendance in real-time. It combines:  
- **Face Detection** (Haar Cascade)  
- **Face Recognition** (K-Nearest Neighbors - KNN)  
- **Attendance Logging** (CSV files)  
- **User Management** (Add/Delete Users)  

**Key Features**:  
✔ **Real-time attendance marking** via webcam  
✔ **Multi-user registration** (stores 10 face samples per user)  
✔ **Daily CSV logs** (auto-generated by date)  
✔ **Simple Flask UI** for interaction

## 📂 **Project Structure**  
```bash

├── static/  
│   ├── faces/                  # Stores registered user faces (format: Name_Roll/)
```

## 🚀 **Setup & Execution**
1. Install Dependencies
```bash
pip install flask opencv-python scikit-learn pandas numpy
```
2. Run the Application
```bash
python app.py
```
Access the web interface at: http://localhost:5000

## 🔧 Core Functions

| Function | Description |
|----------|-------------|
| `extract_faces()` | Detects faces in images using Haar Cascade classifier |
| `identify_face()` | Predicts faces using trained K-Nearest Neighbors (KNN) model |
| `add_attendance()` | Logs recognized faces to daily CSV with timestamp |
| `train_model()` | Trains/retrains KNN classifier on registered user faces |
| `is_face_registered()` | Checks for duplicate faces during new user registration |
